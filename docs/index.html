dotfiles

Void Linux contains the dotfiles for my previous Linux install, Void Linux. I will only be porting updates here every once in a while as I don't use it much.

NixOS contains the dotfiles for my current Linux install, NixOS. This is the active branch.

README

I assume you have all the relevant programs installed either the NixOS way or the Void Linux way.

The top level headers in the boxes are folders in $HOME.

The items with folder icons () are clickable and will open a section at the bottom of the page with files to download and save.

There is a further set of instructions for each program below the links.
